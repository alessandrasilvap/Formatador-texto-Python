# 📝 Formatador de Texto

Aplicação web desenvolvida com **Python** e **Flask** que permite formatar textos de forma rápida e prática.

## 📋 Funcionalidades

- Converter todo o texto para **MAIÚSCULO**;
- Converter todo o texto para **minúsculo**;
- Deixar apenas a primeira letra de cada frase em maiúscula;
- Remover espaços desnecessários do texto.

## 🛠 Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3

## 📁 Estrutura do Projeto

```
projeto/
│
├── app.py
└── templates/
    └── index.html
```

## ▶ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU_USUARIO/formatador-de-texto.git
```

2. Instale o Flask:

```bash
pip install flask
```

3. Execute a aplicação:

```bash
python app.py
```

4. Acesse no navegador:

```
http://127.0.0.1:5000
```
